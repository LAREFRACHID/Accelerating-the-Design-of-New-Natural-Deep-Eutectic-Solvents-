# Accelerating the Design of New Natural Deep Eutectic Solvents with Supervised Variational Autoencoders

This repository contains the complete source code and datasets associated with the manuscript:

**"Accelerating the Design of New Natural Deep Eutectic Solvents with Supervised Variational Autoencoders"**

## Repository Contents

This repository includes:

- All Python scripts used throughout the proposed pipeline.
- The datasets used for melting point, density, and viscosity prediction.
- Descriptor generation and feature selection scripts.
- Supervised Variational Autoencoder (SVAE) implementation.
- Hidden DES recovery validation.
- COCONUT descriptor matching pipeline.
- Machine learning models used for external validation (Random Forest, XGBoost, SVR, and MLP).
- Scripts used to generate all figures and tables presented in the manuscript.

## COCONUT Database

The COCONUT database is **not included** in this repository because of its large size, which exceeds GitHub's file size limits.

It can be  downloaded from this link: https://zenodo.org/records/21303632



or directly from:

https://zenodo.org/record/7019751

After downloading the database, place the corresponding file in the appropriate data directory before running the pipeline.
